caps
Member of the VibrationCapabilities enumeration identifying what capability to query.

Return Value

0 if the capability is not supported, 1 if a single level is supported or a value between 2 and 7 if multiple levels are supported.

Remarks

This function returns the number of vibration steps (a number from 0 to 7) that the device hardware supports for the requested vibration capability.
