maxMessages
Indicates the length of the queue in messages.
maxMessageSize
Indicates the maximum size of each message in the queue.
readAccess
Set to true if messages will be read from the queue, false if messages will be written to the queue. A read/write queue cannot be created.
flags
Flags indicating how the queue will operate.
