Win32Window.GetCapture Method 

[This is preliminary documentation and subject to change.]

Retrieves the handle to the window, if any, that has captured the mouse or stylus input. Only one window at a time can capture the mouse or stylus input.

[Visual Basic]
Public Shared Function GetCapture() As IntPtr
[C#]
public static IntPtr GetCapture();

Return Value

The handle of the capture window associated with the current thread indicates success. NULL indicates that no window in the current thread has captured the mouse.
