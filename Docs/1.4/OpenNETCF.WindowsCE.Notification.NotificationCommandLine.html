NotificationCommandLine Enumeration

[This is preliminary documentation and subject to change.]

Strings passed on the command line when an event occurs that the app has requested via CeRunAppAtEvent.

[Visual Basic]
Public Enum NotificationCommandLine
[C#]

Remarks

Note that some of these strings will be used as the command line *prefix*, since the rest of the command line will be used as a parameter.

Members

Member Name Description
AppRunAtTime String passed on the command line when an app is run as the result of a call to RunAppAtTime.
AppRunToHandleNotification Prefix of the command line when the user requests to run the application that "owns" a notification. It is followed by a space, and the stringized version of the notification handle.
AppRunAfterTimeChange Prefix of the command line when the user requests to run the application when the system time settings are changed.
AppRunAfterSync Prefix of the command line when the user requests to run the application after synchronisation.
AppRunAtAcPowerOn Prefix of the command line when the user requests to run the application when the device is connected to AC power.
AppRunAtAcPowerOff Prefix of the command line when the user requests to run the application when the AC power is disconnected.
AppRunAtNetConnect Prefix of the command line when the user requests to run the application when the device connects to a LAN.
AppRunAtNetDisconnect Prefix of the command line when the user requests to run the application when the device disconnects from a LAN.
AppRunAtDeviceChange Prefix of the command line when the user requests to run the application that "owns" a notification. It is followed by a space, and the stringized version of the notification handle.
AppRunAtIrDiscovery Prefix of the command line when the user requests to run the application when another device is discovered using IR.
AppRunAtRs232Detect Prefix of the command line when the user requests to run the application when a serial port connection is attempted.
AppRunAfterRestore Prefix of the command line when the user requests to run the application after a system restore.
AppRunAfterWakeup Prefix of the command line when the user requests to run the application when the device wakes up from standby.
AppRunAfterTzChange Prefix of the command line when the user requests to run the application when the device time-zone is changed.
AppRunAfterExtendedEvent Prefix of the command line when the user requests to run the application after an extended event.
