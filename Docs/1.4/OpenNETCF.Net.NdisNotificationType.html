NdisNotificationType Enumeration

[This is preliminary documentation and subject to change.]

The NdisNotificationType enumeration defines the types of notifications which can be requested from NDIS.

Members

Member Name Description
NdisResetStart NdisResetStart is set when an adapter is being reset.
NdisResetEnd NdisResetEnd is sent when the reset process on an adapter is complete and the adapter is ready to be rebound.
NdisMediaConnect NdisMediaConnect is set when the communications media, an Ethernet cable for example, is connected to the adapter.
NdisMediaDisconnect NdisMediaDisconnect is set when the communciations media, an Ethernet cable for example, is disconnected from the adapter.
NdisBind NdisBind is set when one or more protocols, TCP/IP typically, is bound to the adapter.
NdisUnbind NdisUnbind is set when the adapter is unbound from one or more protocols.
NdisMediaSpecific NdisMediaSpecific is set when some notification not generally defined for all adapter types occurred.
