The properties of the Adapter class are listed below. For a complete list of Adapter class members, see the Adapter Members topic.

Public Instance Properties

AssociatedAccessPoint Returns the currently-attached SSID for RF Ethernet adapters.
CurrentIpAddress The currently active IP address of the adapter.
CurrentSubnetMask The currently active subnet mask address of the adapter.
Description The descriptive name of the adapter.
DhcpEnabled Indicator of whether DHCP (for IP address assignment from a server), is enabled for the adapter.
DhcpServer The DHCP server from which the IP address was last assigned.
Gateway The active gateway address.
HasWins Indicates the presence of WINS server addresses for the adapter.
Index The index in NDIS' list of adapters where this adapter is found.
IsWireless Returns a Boolean indicating if the adapter is an RF Ethernet adapter.
IsWirelessZeroConfigCompatible Returns a Boolean indicating if the adapter is supported by WZC.
LeaseExpires The date/time at which the IP address lease from the DHCP server will expire (at which time the adapter will have to contact the server to renew the lease or stop using the IP address).
LeaseObtained The date/time at which the IP address lease was obtained from the DHCP server.
MacAddress The hardware address associated with the adapter. For Ethernet-based adapters, including RF Ethernet adapters, this is the Ethernet address.
Name The NDIS/driver assigned adapter name.
NearbyAccessPoints Returns a list of the SSID values which the adapter can currently 'hear'.
NearbyPreferredAccessPoints Returns the list of preferred SSID values which the adapter is currently assigned, but also updates the signal strengths to their current values. Otherwise, the signal strengths are not really valid.
PreferredAccessPoints Returns the list of preferred SSID values which the adapter is currently assigned. Note that none of these might be within range, however.
PrimaryWinsServer The IP address of the primary WINS server for the adapter.
SecondaryWinsServer The IP address of the secondary WINS server for the adapter.
SignalStrength Returns a SignalStrength class representing the current strength of the signal.
SignalStrengthInDecibels Returns the strength of the RF Ethernet signal being received by the adapter, in dB.
Type The adapter type. Adapters can be standard Ethernet, RF Ethernet, loopback, dial-up, etc.
WZCFallbackEnabled Enables or disables WZC Fallback for the current adapter
