Remarks

This interface allows an application to capture a message before it is dispatched to a control or form.

A class that implements the IMessageFilter interface can be added to the application's message pump to filter out a message or perform other operations before the message is dispatched to a form or control. To add the message filter to an application's message pump, use the AddMessageFilter method in the ApplicationEx class.
