Process.Exited Event

[This is preliminary documentation and subject to change.]

Occurs when a process exits.

[Visual Basic]
Public Event Exited As EventHandler
[C#]
public event EventHandler Exited;

Remarks

Note that in this partial implementation the Exited event will only be raised when a Process is explictly Killed. The Process class will not watch created processes unless you use the WaitForExit method.
