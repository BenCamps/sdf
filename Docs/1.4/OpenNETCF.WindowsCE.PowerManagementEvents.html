PowerManagement Events

The events of the PowerManagement class are listed below. For a complete list of PowerManagement class members, see the PowerManagement Members topic.

Public Static (Shared) Events

Boot Fired when the device receives a notification that it has booted
PowerCritical Fired when the device receives a notification that a critical power state has been reached
PowerDown Fired when the device receives a notification that it has been powered down/suspends
PowerIdle Fired when the device receives a notification that it has entered an idle power state
PowerUp Fired when the device receives a notification that it has been powered up/wakes
