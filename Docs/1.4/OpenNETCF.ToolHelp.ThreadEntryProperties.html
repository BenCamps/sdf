ThreadEntry Properties

The properties of the ThreadEntry class are listed below. For a complete list of ThreadEntry class members, see the ThreadEntry Members topic.

Public Instance Properties

BasePriority Initial priority level assigned to a thread, which can be a value from 0 to 255.
CurrentProcessID Process identifier where the thread is executing.
DeltaPriority Change in the priority level of a thread. This value is a signed delta from the base priority level assigned to the thread.
OwnerProcessID Identifier of the process that created the thread. The contents of this member can be used by Win32 API elements.
ThreadID Identifier of the thread. This identifier is compatible with the thread identifier returned by the CreateProcess function.
Usage Number of references to the thread. A thread exists as long as its usage count is nonzero. As soon as its usage count becomes zero, a thread terminates.
