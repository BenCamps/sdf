Crypto.CryptVerifySignature Method 

[This is preliminary documentation and subject to change.]

This function verifies the signature of a hash object. Before calling this function, the CryptCreateHash function must be called to get a handle to a hash object. The CryptHashData function or CryptHashSessionKey function is then used to add the data or session keys to the hash object. After the call to the CryptVerifySignature function has been completed, only the CryptDestroyHash function can be called using the hHash handle.
