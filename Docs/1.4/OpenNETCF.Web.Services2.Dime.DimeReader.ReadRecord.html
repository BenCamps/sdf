DimeReader.ReadRecord Method 

[This is preliminary documentation and subject to change.]

Returns the next record in the DIME message in read-only mode. If the DimeReader is closed then an InvalidOperationException will occur. When the end of the message is reached ReadRecord will return null. ReadRecord is a blocking i/o call in that the DIME record header is read before the method returns.

[Visual Basic]
Public Function ReadRecord() As DimeRecord
[C#]
public DimeRecord ReadRecord();

Return Value

A read-only DimeRecord, or null if it's the end of the DIME message. If the record has TNF=None and it is the last record in the message null is also returned.
