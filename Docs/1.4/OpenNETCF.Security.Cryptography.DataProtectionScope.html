Specifies the scope of the data protection to be applied by the Protect method

[Visual Basic]
Public Enum DataProtectionScope
[C#]
public enum DataProtectionScope

Remarks

This enumeration is used with the Protect and Unprotect methods to protect data through encryption.

Members

Member Name Description
CurrentUser Specifies that the protected data is associated with the current user. Only threads running under the current user context can unprotect the data.
LocalMachine Specifies that the protected data is associated with the machine context. Any process running under Admin credentials on the computer can unprotect data.
