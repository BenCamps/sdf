buffer
Caller-allocated byte array for the data.
bufferSize
Number of bytes in the caller-allocated buffer.
bytesRead
Return value indicating number of bytes actually read and stored in the buffer.
timeout
Time-out value in ms for the read. If no messages arrive before the time-out occurs, the number of bytes read will be zero.
readFlags
Will be set to AlertMsg if the message was written to the queue with that flag set. Zero, otherwise.

Return Value

True is returned on a successful read; false otherwise.
