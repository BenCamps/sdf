Returns a string which identifies the device platform.

[Visual Basic]
Public Shared ReadOnly Property PlatformName As String
[C#]
public static string PlatformName {get;}

Remarks

Valid values include:-

  • PocketPC - Pocket PC device or Emulator
  • SmartPhone - Smartphone 2003 Device or Emulator
  • CEPC platform - Windows CE.NET Emulator
Additional platform types will have other names. Useful when writing library code targetted at multiple platforms.
