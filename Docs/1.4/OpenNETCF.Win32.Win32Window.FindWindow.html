Win32Window.FindWindow Method 

[This is preliminary documentation and subject to change.]

This function retrieves the handle to the top-level window whose class name and window name match the specified strings.

Parameters

className

wndName
The window name (the window's title). If this parameter is NULL, all window names match.

Return Value

A handle to the window that has the specified class name and window name indicates success. NULL indicates failure.
