TimeZoneCollection.Initialize Method (Int32)

[This is preliminary documentation and subject to change.]

Use the Initialize method with the gmtOffset parameter to create a list of timezones which have a specific offset value. For example, for all timezones at GMT-5:00, pass -300 (minutes). To get all of the time zones known to the OS, pass ALL_TIMEZONES_LIST or call the method with no parameters.
