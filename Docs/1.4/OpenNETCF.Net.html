ConnectionManager A class that handles opening and closing of network connections.

It is important to remember that when connecting asynchronous, you must poll the connection status in your code. 

bool connected = false;
ConnectionManger connMgr = new ConnectionManager();
connMgr.Connect(true, ConnectionMode.Asynchronous);
while(!connected)
{
    if(connMgr.Status != ConnectionStatus.Connected)
    {
        Thread.Sleep(1000);
        continue;
    }
    connected = true;
}
// Now we are connected and can do something...

It is also important to note that creating a non-exclusive connection will result in your application being unable to programmatically disconnect.
