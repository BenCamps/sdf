Process.WaitForExit Method ()

[This is preliminary documentation and subject to change.]

Instructs the Process component to wait indefinitely for the associated process to exit.

[Visual Basic]
Overloads Public Sub WaitForExit()
[C#]
public void WaitForExit();

Remarks

WaitForExit is used to make the current thread wait until the associated process terminates. This overload of Process.WaitForExit instructs the Process component to wait an infinite amount of time for the process to exit. This can cause an application to stop responding.
